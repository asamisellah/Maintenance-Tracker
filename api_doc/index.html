Maintenance Tracker API

Maintenance Tracker App is an application that provides users with the ability to reach out to operations or repairs department regarding repair or maintenance requests and monitor the status of their request.

Requests

You can create, view, update and delete requests once logged in. NOTE: You must include the access token in the header else a 401 or 404 response will be returned.

Requests Resources

You can view a list of all the requests you made. You cannot view another user’s requests.
